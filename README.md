# ボタンリスト

## 概要

音声ファイルをアップロードするとボタンが作成されて、共有できるアプリ。

## 使い方

-   音声ファイルをアップロードして自動的にボタンを作成
-   ハートマークを押すことでお気に入りに登録

## 開発環境

-   Ubuntu 18.04
-   Ruby 3.03
-   PostgrSQL

## 使用技術

-   Ruby on Rails 6.0.4
-   jQuery

## 使用した gem

-   devise
-   mini_magick
-   bootstrap
-   jquery-rails
-   kaminari
-   aws-sdk-s3
